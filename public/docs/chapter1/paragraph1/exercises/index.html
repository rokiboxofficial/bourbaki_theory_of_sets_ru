    Упражнения#

    Упражнение 1)#

    1.1#

    Пусть $\boldsymbol{A}_1, \boldsymbol{A}_2, \dots, \boldsymbol{A}_n$ — формативная конструкция теории $\mathcal{T}$, где $\boldsymbol{A}_n$ — единственное знакосочетание второго рода.

    По определению формативной конструкции, для каждого знакосочетания $\boldsymbol{A}$ из последовательности $\boldsymbol{A}_1, \boldsymbol{A}_2, \dots, \boldsymbol{A}_n$ должно выполняться одно из условий. Покажем, что для $\boldsymbol{A}_n$ не выполняется ни одно из них:

    а) $\boldsymbol{A}_n$ является знакосочетанием второго рода, значит, по определению знакосочетания второго рода, $\boldsymbol{A}_n$ не сводится к одной букве. $\boldsymbol{A}_n$ — не буква.

    б, в, г) $\boldsymbol{A}_n$ — единственное знакосочетание второго рода в последовательности $\boldsymbol{A}_1, \boldsymbol{A}_2, \dots, \boldsymbol{A}_n$, значит, никакое другое знакосочетание второго рода не предшествует $\boldsymbol{A}_n$ в $\boldsymbol{A}_1, \boldsymbol{A}_2, \dots, \boldsymbol{A}_n$.

    д) теория $\mathcal{T}$ не содержит специальных знаков.

    Ясно, что не существует такой формативной конструкции теории $\mathcal{T}$. Очевидно, что теория $\mathcal{T}$ не содержит соотношений.

    1.2#

    Пусть $\boldsymbol{A}_1, \boldsymbol{A}_2, \dots, \boldsymbol{A}_n$ — формативная конструкция теории $\mathcal{T}$, где $\boldsymbol{A}_n$ — единственное знакосочетание первого рода, не сводящееся к одной букве.

    По определению формативной конструкции, для каждого знакосочетания $\boldsymbol{A}$ из последовательности $\boldsymbol{A}_1, \boldsymbol{A}_2, \dots, \boldsymbol{A}_n$ должно выполняться одно из условий. Покажем, что для $\boldsymbol{A}_n$ не выполняется ни одно из них:

    а) по определению $\boldsymbol{A}_1, \boldsymbol{A}_2, \dots, \boldsymbol{A}_n$, знакосочетание $\boldsymbol{A}_n$ не сводится к одной букве. $\boldsymbol{A}_n$ — не буква.

    б, в, г) согласно 1.1, теория $\mathcal{T}$ не содержит соотношений, значит, ни одно знакосочетание второго рода не предшествует $\boldsymbol{A}_n$ в $\boldsymbol{A}_1, \boldsymbol{A}_2, \dots, \boldsymbol{A}_n$.

    д) теория $\mathcal{T}$ не содержит специальных знаков.

    Ясно, что не существует такой формативной конструкции теории $\mathcal{T}$. Очевидно, что единственные знакосочетания теории $\mathcal{T}$, являющиеся термами, суть знакосочетания, сводящиеся к одной букве.

    1.3#

    Из 1.1 и 1.2 следует, что теория $\mathcal{T}$ не содержит соотношений, а единственные знакосочетания теории $\mathcal{T}$, являющиеся термами, суть знакосочетания, сводящиеся к одной букве.

    Упражнение 2)#

    1.1#

    Пусть $\boldsymbol{A}_1, \boldsymbol{A}_2, \dots, \boldsymbol{A}_n$ - формативная конструкция теории $\mathcal{T}$, где $\boldsymbol{A}_n$ единственное знакосочетание, которое содержит знак $\square$, не связанный с одним и только одним знаком $\tau$ слева от него. По определению формативной конструкии, для каждого знакосочетания $\boldsymbol{A}$ из $\boldsymbol{A}_1, \boldsymbol{A}_2, \dots, \boldsymbol{A}_n$ должно выполняться одно из условий. Покажем, что для $\boldsymbol{A}_n$ не выполняется ни одно из условий:

    а) $\boldsymbol{A}_n$ содержит знак $\square$, значит, $\boldsymbol{A}_n$ не буква.

    б, в, д) эти условия сводятся к тому, что $\boldsymbol{A}_n$ это знакосочетание, полученное написанием слева направо знака веса n, отличного от $\square$, и n знакосочетаний из предществующих $\boldsymbol{A}_n$ знакосочетаний. Так как в последовательности $\boldsymbol{A}_1, \boldsymbol{A}_2, \dots, \boldsymbol{A}_n$, знакосочетание $\boldsymbol{A}_n$ является единственным знакосочетанием, которое содержит знак $\square$, не связанный с одним и только одним знаком $\tau$ слева от него, то в предществующих $\boldsymbol{A}_n$ знакосочетаниях, такого знака быть не может. Очевидно, что добавление знака, отличного от $\square$, не влияет на связи, значит, эти условия не выполняются

    г) По условию, $\boldsymbol{A}_n$ есть $\tau_x(B)$, где $\boldsymbol{B}$ - знакосочетание второго рода, предшествующее $\boldsymbol{A}_n$. Так как в последовательности $\boldsymbol{A}_1, \boldsymbol{A}_2, \dots, \boldsymbol{A}_n$, знакосочетание $\boldsymbol{A}_n$ является единственным знакосочетанием, которое содержит знак $\square$, не связанный с одним и только одним знаком $\tau$ слева от него, то $\boldsymbol{B}$ не содержит такого знака. Знакосочетание $\tau_x(B)$, по определению, не могло создать связи, отличные от связей $\square$ со знаком $\tau$, который находится слева от $\square$, и для каждого $\square$ такая связь одна и только одна.

    Ясно, что не существует такой формативной конструкции теории $\mathcal{T}$. Пусть $\boldsymbol{A}$ - терм или соотношение теории $\mathcal{T}$. Каждый знак $\square$, если он имеется в $\boldsymbol{A}$, связан с одним и только одним знаком $\tau$, расположенным слева от него
